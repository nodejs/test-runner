# Test Runner team

This repository is for proposing & planning upcoming features, tracking meetings, and triaging issues.

This repository is NOT for reporting issues with existing features (report those in https://github/com/nodejs/node/issues).

Discussions: discuss nuances to feature proposals

Issues: plan meetings & specific, targeted discussions

Pull Requests: propose & plan features, add minuting minutes
